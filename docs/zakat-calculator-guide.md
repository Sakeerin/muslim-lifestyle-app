# Zakat Calculator — คู่มือการใช้งาน / Usage Guide

## ภาพรวม (Overview)

เครื่องมือคำนวณซะกาตช่วยให้คุณคำนวณภาระซะกาตประจำปีตามหลักศาสนาอิสลาม
The Zakat Calculator helps you calculate your annual Zakat obligation according to Islamic principles.

**URL:** `/zakat`
**เข้าถึงได้จาก:** Quick Access บนหน้าแรก → "คำนวณซะกาต" / Sidebar nav → "Zakat"

---

## หลักการคำนวณ (Calculation Rules)

| รายการ | ค่า |
|--------|-----|
| นิศอบ — มาตรฐานทอง | 87.48 กรัม |
| นิศอบ — มาตรฐานเงิน | 612.36 กรัม |
| อัตราซะกาต | 2.5% (1/40) ของทรัพย์สินสุทธิ |
| เงื่อนไขเฮาล์ | ทรัพย์สินต้องถือครองครบ 1 ปีจันทรคติ |

**สูตรการคำนวณ:**
```
ทรัพย์สินรวม  = เงินสด + ทอง + เงิน + ธุรกิจ + การลงทุน + เงินรับได้
ทรัพย์สินสุทธิ = ทรัพย์สินรวม − หนี้สิน
ซะกาตที่ต้องจ่าย = ทรัพย์สินสุทธิ × 0.025  (ถ้า สุทธิ ≥ นิศอบ)
```

## ขั้นตอนการใช้งาน (Step-by-Step)

### ขั้นที่ 1 — เลือกมาตรฐานนิศอบ

- **ทอง (Gold)** — ใช้ราคาทอง 87.48 กรัม เป็นเกณฑ์ (นิยมใช้มากกว่า)
- **เงิน (Silver)** — ใช้ราคาเงิน 612.36 กรัม เป็นเกณฑ์ (ให้เกณฑ์ต่ำกว่า)

### ขั้นที่ 2 — ตรวจสอบราคาโลหะ

ระบบดึงราคาสด (USD/กรัม) จาก **metals.live** โดยอัตโนมัติ
แสดง badge `ราคาสด · USD/กรัม` เมื่อโหลดสำเร็จ

- **ปรับแมนนวล:** พิมพ์ราคาในช่องเพื่อใช้ราคาสกุลเงินท้องถิ่น
- **กลับราคาอัตโนมัติ:** กดไอคอน ↺ หรือลิงก์ "ใช้ราคาสดอีกครั้ง"

### ขั้นที่ 3 — กรอกทรัพย์สินที่ต้องเสียซะกาต

| ช่อง | รายละเอียด |
|------|-----------|
| เงินสดและบัญชีธนาคาร | ยอดรวมในบัญชีทุกประเภท |
| ทองคำ (มูลค่าตลาด) | มูลค่าปัจจุบันของทอง/ทองแท่ง |
| เงิน (มูลค่าตลาด) | มูลค่าปัจจุบันของเงินที่ถือครอง |
| สินค้าคงคลังธุรกิจ | มูลค่าสินค้าที่มีไว้ขาย |
| การลงทุนและหุ้น | มูลค่าตลาดของหุ้น/กองทุน |
| เงินที่รับได้จากผู้อื่น | หนี้ที่คาดว่าจะได้รับคืน |

### ขั้นที่ 4 — กรอกหนี้สิน

- **หนี้สินทันที** — เฉพาะหนี้ที่ครบกำหนดภายในปีจันทรคตินี้เท่านั้น
- ไม่รวมผ่อนบ้าน/รถที่ยังมีหลายปี (ใส่เฉพาะงวดที่ค้างปีนี้)

### ขั้นที่ 5 — อ่านผลการคำนวณ

ผลลัพธ์อัปเดตแบบ real-time ทุกครั้งที่เปลี่ยนค่า:

| ผลลัพธ์ | ความหมาย |
|---------|---------|
| ทรัพย์สินรวม | ผลรวมทุกช่องทรัพย์สิน |
| หนี้สินรวม | ยอดหนี้ที่กรอก |
| ทรัพย์สินสุทธิ | ทรัพย์สิน − หนี้ (ไม่ติดลบ) |
| เกณฑ์นิศอบ | ราคา × น้ำหนักโลหะ |
| 🟢 ต้องจ่ายซะกาต | สุทธิ ≥ นิศอบ |
| ⚫ ต่ำกว่านิศอบ | สุทธิ < นิศอบ — ไม่ต้องจ่าย |

## ตัวอย่างการคำนวณ (Worked Examples)

### ตัวอย่างที่ 1 — ต้องจ่ายซะกาต (Eligible)

**สมมติ:** ราคาทองคือ 3,200 USD/troy oz → 102.88 USD/กรัม

| ทรัพย์สิน | มูลค่า (USD) |
|-----------|-------------|
| เงินสด | 15,000 |
| ทองคำ | 5,000 |
| การลงทุน | 10,000 |
| หนี้สิน | 3,000 |

**การคำนวณ:**
```
ทรัพย์สินรวม   = 15,000 + 5,000 + 10,000 = 30,000 USD
ทรัพย์สินสุทธิ = 30,000 − 3,000 = 27,000 USD
นิศอบ (ทอง)   = 87.48 × 102.88 = 8,999.98 USD

27,000 ≥ 8,999.98 → ✅ ต้องจ่ายซะกาต

ซะกาตที่ต้องจ่าย = 27,000 × 0.025 = 675.00 USD
```

---

### ตัวอย่างที่ 2 — ต่ำกว่านิศอบ (Not Eligible)

**สมมติ:** ราคาทอง 102.88 USD/กรัม

| ทรัพย์สิน | มูลค่า (USD) |
|-----------|-------------|
| เงินสด | 5,000 |
| การลงทุน | 2,000 |
| หนี้สิน | 1,000 |

**การคำนวณ:**
```
ทรัพย์สินรวม   = 5,000 + 2,000 = 7,000 USD
ทรัพย์สินสุทธิ = 7,000 − 1,000 = 6,000 USD
นิศอบ (ทอง)   = 87.48 × 102.88 = 8,999.98 USD

6,000 < 8,999.98 → ❌ ต่ำกว่านิศอบ ไม่ต้องจ่าย
```

---

### ตัวอย่างที่ 3 — มาตรฐานเงิน (Silver Standard)

**สมมติ:** ราคาเงิน 1.05 USD/กรัม

```
นิศอบ (เงิน) = 612.36 × 1.05 = 642.98 USD

ผู้ที่มีทรัพย์สินสุทธิ 700 USD → ต้องจ่ายซะกาต
ซะกาต = 700 × 0.025 = 17.50 USD
```

> **หมายเหตุ:** มาตรฐานเงินให้เกณฑ์ต่ำกว่ามาก ทำให้คนจำนวนมากขึ้นต้องจ่ายซะกาต

---

### ตัวอย่างที่ 4 — ราคาในสกุลเงินท้องถิ่น (Thai Baht)

**สมมติ:** ราคาทองไทย 3,500 บาท/กรัม

1. กรอก `3500` ในช่องราคาทองต่อกรัม (ระบบจะถามว่าต้องการแก้ไขเอง)
2. กรอกทรัพย์สินทั้งหมดใน **หน่วยบาท**

```
นิศอบ = 87.48 × 3,500 = 306,180 บาท

ถ้าทรัพย์สินสุทธิ = 400,000 บาท
ซะกาต = 400,000 × 0.025 = 10,000 บาท
```

## ฟีเจอร์หลัก (Key Features)

| ฟีเจอร์ | รายละเอียด |
|---------|-----------|
| **ราคาสด** | ดึงจาก metals.live ทุก 1 ชั่วโมง (cached) |

| **Manual override** | พิมพ์ราคาเองได้ กด ↺ เพื่อกลับ |

| **Reset** | ปุ่มรีเซ็ตทุกช่องกลับเป็น 0 |
| **Dark mode** | รองรับธีมสว่าง/มืด |

## API Endpoint (สำหรับนักพัฒนา)

```
GET /api/metal-prices
```

**Response:**
```json
{
  "goldPerGram": 102.8837,
  "silverPerGram": 1.0521,
  "currency": "USD",
  "source": "metals.live",
  "updatedAt": "2026-03-13T10:00:00.000Z"
}
```

- ดึงจาก `https://metals.live/api/spot`
- แปลงจาก USD/troy oz → USD/gram (÷ 31.1034768)
- Cache `revalidate: 3600` (1 ชั่วโมง)

## โค้ดตัวอย่าง — ใช้ `calculateZakat` โดยตรง

```typescript
import { calculateZakat, calculateNisab } from "@/lib/zakat-utils";

// ตัวอย่าง: คำนวณซะกาตจากข้อมูลที่กำหนด
const result = calculateZakat(
  {
    cash: 15000,
    goldValue: 5000,
    silverValue: 0,
    business: 0,
    investments: 10000,
    receivables: 0,
  },
  { debts: 3000 },
  { standard: "gold", pricePerGram: 102.88 }
);

console.log(result.isEligible);   // true
console.log(result.zakatDue);     // 675.00
console.log(result.netWealth);    // 27000
console.log(result.nisabThreshold); // 8999.98...

// คำนวณนิศอบเท่านั้น
const nisab = calculateNisab("silver", 1.05);
console.log(nisab); // 642.98
```

## ข้อจำกัดและข้อควรระวัง

1. **ราคาเป็น USD** — ราคาที่ดึงอัตโนมัติเป็น USD/กรัม หากต้องการสกุลท้องถิ่น ให้ป้อนราคาเอง
2. **ไม่ track เฮาล์** — แอปไม่บันทึกว่าถือครองทรัพย์สินครบปีหรือยัง ต้องตรวจสอบเอง
3. **การประมาณการเท่านั้น** — สำหรับสถานการณ์ซับซ้อน ควรปรึกษาอาลิมผู้รู้
